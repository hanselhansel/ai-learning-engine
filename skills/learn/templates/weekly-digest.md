# Weekly Learning Digest Template

## Format

Generate this digest every 5 sessions OR every 7 calendar days (whichever comes first).

```
╔══════════════════════════════════════════════╗
║          WEEKLY LEARNING DIGEST              ║
╠══════════════════════════════════════════════╣
║  Week: [WEEK_NUMBER]                         ║
║  Sessions: [START]-[END] ([COUNT] total)     ║
║  Dates: [START_DATE] - [END_DATE]            ║
║  Streak: [CURRENT_STREAK] days               ║
╠══════════════════════════════════════════════╣
║  CONCEPTS COVERED                            ║
║  • [Concept 1] — [1-line summary]            ║
║  • [Concept 2] — [1-line summary]            ║
║  • ...                                       ║
╠══════════════════════════════════════════════╣
║  QUIZ PERFORMANCE                            ║
║  Average: [AVG]/10                           ║
║  Trend: [↑/↓/→] from last week              ║
║  Strongest: [TERM] ([SCORE])                 ║
║  Weakest: [TERM] ([SCORE])                   ║
╠══════════════════════════════════════════════╣
║  SM-2 TERMS DUE NEXT WEEK                   ║
║  • [Term 1] (due session [N])                ║
║  • [Term 2] (due session [N])                ║
║  • ...                                       ║
╠══════════════════════════════════════════════╣
║  SPICY QUESTIONS TO THINK ABOUT             ║
║  1. [Thought-provoking question about        ║
║     upcoming concepts]                       ║
║  2. [Question connecting this week's         ║
║     learning to learner's objective]         ║
║  3. [Question previewing next week]          ║
╠══════════════════════════════════════════════╣
║  NEXT WEEK PREVIEW                           ║
║  Day [X]: [Topic]                            ║
║  Day [Y]: [Topic]                            ║
║  Day [Z]: [Topic]                            ║
╚══════════════════════════════════════════════╝
```

## Save Location
Save to `sessions/digests/week-XX.md`

## Trigger Rules
- Every 5 sessions (session 5, 10, 15, 20...)
- OR every 7 calendar days since last digest
- Whichever comes first
- Generate at the END of the triggering session (during Step 6 close)
