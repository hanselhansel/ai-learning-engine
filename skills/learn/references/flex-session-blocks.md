# Flex Session Modes & Block Composition

## Mode Selection
At session open, ask: "How much time do you have?"
- **Micro** (5 min): Coffee-break review. SM-2 only.
- **Quick** (15-20 min): Review + light practice. No new concepts.
- **Standard** (50-65 min): Full learning cycle. Default mode.
- **Deep Dive** (90-120+ min): Standard + teach-back + concept linking.
- **Synthesis** (60-75 min): Every 5th session. Integration challenge.

## Mode-Specific Rules

### Micro (5 min)
- NO new concepts taught
- 3 SM-2 terms + 1 interleaved question + velocity check
- Skip if no terms are due for review

### Quick (15-20 min)
- NO new concepts taught
- Full SM-2 quiz (up to 10 terms) + interleaved practice
- Include 1 exercise from spaced exercise queue

### Standard (50-65 min)
- DEFAULT mode. Full learning cycle.
- Max 7 new concepts (fewer if compression active)
- Teach-back triggers occasionally (every 3rd Standard session)

### Deep Dive (90-120+ min)
- All Standard blocks PLUS teach-back, concept linking, community
- Session variety (C4): occasionally swap portfolio for debate/case study/reverse quiz

### Synthesis (every 5th session, 60-75 min)
- Replaces Standard on sessions 5, 10, 15, 20...
- All SM-2 terms from the week
- Integration challenge replaces lesson + practice
- 2-min pitch of the week's big idea

## Completion Criteria (for learn-end)
A mode is "complete" when all its required blocks (Y) have been executed.
Partial completion = save checkpoint with last completed block.
